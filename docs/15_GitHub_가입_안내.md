# 15. GitHub 가입 안내

## 1. 목적

Git 실습을 진행하기 전에 교육생이 GitHub 개인 계정을 생성하고 이메일 인증을 완료하도록 안내합니다.

---

## 2. 사전 준비

| 항목 | 설명 |
|---|---|
| 이메일 주소 | GitHub 가입 및 인증에 사용할 이메일 |
| 브라우저 | Chrome 또는 Edge 권장 |
| 비밀번호 | 개인이 관리 가능한 안전한 비밀번호 |

---

## 3. 가입 절차

1. GitHub 사이트에 접속합니다.
2. `Sign up`을 선택합니다.
3. 이메일 주소를 입력합니다.
4. 비밀번호를 설정합니다.
5. 사용자 이름을 입력합니다.
6. 화면 안내에 따라 가입을 완료합니다.
7. 이메일 인증 메일을 확인합니다.
8. 이메일 인증을 완료합니다.

---

## 4. 이메일 인증이 필요한 이유

GitHub에서는 기본 기능 사용을 위해 이메일 인증이 필요합니다.

이메일 인증이 완료되지 않으면 저장소 생성 등 일부 기본 작업이 제한될 수 있습니다.

---

## 5. 가입 완료 기준

아래 조건을 만족하면 GitHub 가입 준비가 완료된 것입니다.

- GitHub 로그인 가능
- 이메일 인증 완료
- 프로필 화면 접근 가능
- 저장소 생성 화면 접근 가능
