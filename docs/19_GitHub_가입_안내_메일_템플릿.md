# 19. GitHub 가입 안내 메일 템플릿

## 목적

Git 교육 전에 교육생에게 GitHub 가입과 저장소 초대 수락을 안내하기 위한 메일 예시입니다.

---

## 메일 제목

```text
Git 실습 교육을 위한 GitHub 가입 및 저장소 초대 안내
```

---

## 메일 본문

```text
안녕하세요.

Git 실습 교육 진행을 위해 GitHub 가입과 저장소 초대 수락이 필요합니다.

아래 순서대로 진행해 주세요.

1. GitHub 계정을 생성합니다.
2. 이메일 인증을 완료합니다.
3. 저장소 초대 메일을 확인합니다.
4. 초대 수락 후 저장소 접근 여부를 확인합니다.
5. 교육 당일 Git Bash와 VS Code를 사용할 수 있도록 준비합니다.

교육 전까지 저장소 화면에 접근되는지 확인해 주세요.

감사합니다.
```
