# 13. 캡처 필요 이미지 목록

## 1. 목적

교육생이 GitHub, Git Bash, VS Code 화면을 보면서 따라 할 수 있도록 실제 화면 캡처를 추가합니다.

이미지는 직접 캡처한 화면을 사용하고, 계정명이나 개인 정보가 불필요하게 노출되지 않도록 정리한 뒤 추가합니다.

## 7. 캡처 기준

| 기준 | 설명 |
|---|---|
| 화면 크기 | 글자가 읽히도록 확대 후 캡처 |
| 범위 | 한 이미지에는 하나의 설명 대상만 포함 |
| 표시 | 클릭 위치나 확인 위치는 빨간 박스 또는 화살표 표시 |
| 정리 | 불필요한 브라우저 탭과 알림 제거 |
| 파일명 | 숫자_내용 형식으로 통일 |
