# Commit Message 예시

## 기본 형식

```text
type(scope): 변경 내용
```

## 실무 예시

```text
docs(readme): 교육 개요 설명 추가
feat(sql): 사용자 접속 통계 쿼리 추가
feat(python): 접속 로그 집계 코드 추가
docs(report): 분석 결과 요약 추가
fix(sql): 집계 기준일 조건 수정
chore(folder): 실습 폴더 구조 정리
refactor(python): 집계 로직 함수화
test(data): 필수 컬럼 검증 로직 추가
```

## 최소 Type

| Type | 기준 |
|---|---|
| feat | 새 기능, 새 분석, 새 산출물 |
| fix | 오류 수정 |
| docs | 문서 수정 |
| chore | 정리, 설정, 유지보수 |
| refactor | 결과 변화 없는 구조 개선 |
| test | 검증 코드 추가 또는 수정 |
